> why? 성능 유지 및 읽기 쉽게 - 유지 관리해야 할 사람이 마치 도끼 살인자일 것 같다고 예상하고...

> 3)3.Efficiency - 빠르고 효율적인가
> 4)4.Length - 코드 길이(상대적으로 덜 중요)
